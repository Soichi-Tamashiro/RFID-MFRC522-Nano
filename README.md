# RFID-MFRC522-Nano
Uso de la placa MFRC522 con arduino Nano
## Conexiones Arduino Nano con MFRC522(RFID 13.56MHz)##
  * RST_PIN  5
  * SS_PIN  10
  * MISO    11
  * MOSI    12
  * SCK     13
  * VCC     3.3V
  * GND     GND
